# ABS Tema

Tema WordPress personalizado para ABS Global, baseado no Hello Elementor.

## Informações do Tema

- **Nome:** ABS Tema
- **Versão:** 1.0.0
- **Autor:** Fale Agência Digital
- **Website:** https://faleagencia.digital
- **Tema Pai:** Hello Elementor
- **Licença:** GPL v3 ou posterior

## Descrição

ABS Tema é um tema filho (child theme) do Hello Elementor, desenvolvido especialmente para ABS Global. Mantém toda a leveza e flexibilidade do Hello Elementor, permitindo personalizações sem perder as atualizações do tema pai.

## Características

- ✅ Baseado no Hello Elementor
- ✅ Totalmente compatível com Elementor
- ✅ Leve e otimizado para performance
- ✅ Responsivo e mobile-friendly
- ✅ Pronto para tradução
- ✅ Suporte a logo customizado
- ✅ Suporte a cores customizadas
- ✅ Suporte a imagem de cabeçalho
- ✅ Compatível com WooCommerce
- ✅ Acessível (accessibility-ready)

## Requisitos

- WordPress 6.0 ou superior
- PHP 7.4 ou superior
- Tema Hello Elementor instalado
- Plugin Elementor (recomendado)

## Instalação

1. Certifique-se de que o tema Hello Elementor está instalado
2. Faça upload da pasta `abs-tema` para `/wp-content/themes/`
3. Ative o tema em **Aparência → Temas**

## Estrutura de Arquivos

```
abs-tema/
├── style.css           # Estilos do tema e informações do header
├── functions.php       # Funções e hooks do tema
├── screenshot.png      # Imagem de preview do tema
├── README.md          # Este arquivo
└── languages/         # Arquivos de tradução (futuro)
```

## Customização

### Adicionando Estilos CSS

Adicione seus estilos customizados no arquivo `style.css` após a linha de importação do tema pai.

### Adicionando Funcionalidades PHP

Adicione suas funções customizadas no arquivo `functions.php` no final do arquivo.

### Cores Padrão

O tema define variáveis CSS customizadas:

```css
:root {
	--abs-primary-color: #0073aa;
	--abs-secondary-color: #005177;
	--abs-accent-color: #00a0d2;
}
```

Você pode modificar essas cores no arquivo `style.css`.
