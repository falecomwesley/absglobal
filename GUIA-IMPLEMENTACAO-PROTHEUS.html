  • api/ecommerce/v1/orderChangeStatus

Configuração essencial

  1. Definir URL real da API Protheus no admin do plugin.
  2. Configurar autenticação (Basic ou OAuth2).
  3. Ajustar endpoints/params no tab Advanced se necessário.
  4. Executar teste de conexão e sync ponta a ponta.
